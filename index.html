Hello!


My name is Bianca Romana Knülle and I'm a media computer science student at the Stuttgart Media University, Germany. After my apprenticeship as an audio-visual media designer, I rediscovered programming for myself. Since then I try to educate myself in all areas of computer science, with special interest in game development, animation and graphic design.
I would describe myself as a creative, determined and curious person. In the future, I look forward to gaining experience and acquiring new knowledge of media computer science.

Games

GAME MASTER

GameMaster is a Bullet-hell arena multiplayer pixel-art game, where three players have to win against one Game Master by fighting enemys. The Game Master must eliminate all players through spawning enemys. The game was created with Java, LibGDX, Box2D and Kryonet.

PROJECT MASSIVE

Project Massive is a third person open playground world with obstacles and persistent moveable objects for the players to interact with them. Players can show character-emotes and chat via textbubbles. This game will be created in Unreal Engine 5 with C++ and Docker network.
